## 1. Overview

This SOP describes the process for requesting a new laptop, replacement laptop, or laptop upgrade. All hardware requests must go through the official IT asset request process to ensure proper tracking, budgetary approval, and provisioning.

## 4. Laptop Request Process

### 4.1 New Joiner Laptop (Pre-Joining)

This is handled automatically by HR and IT during the onboarding process:
```
1. HR initiates onboarding request in IT portal (30 days before joining date)
2. IT assigns and configures the laptop based on grade and role
3. Laptop is ready for collection on Day 1 at IT Help Desk (Floor 2)
4. New employee signs the IT Asset Acknowledgement Form on Day 1
```

---

### 4.2 Replacement / Upgrade / Additional Device Request

```
Step 1: Log in to IT Help Desk portal
        → helpdesk.company.com

Step 2: Click "New Request" → Category: "Hardware" → Sub-category: "Laptop Request"

Step 3: Fill in the request form:
        - Request Type (Replacement / Upgrade / Additional)
        - Reason for request
        - Current laptop serial number (if replacing)
        - Business justification

Step 4: Submit the form

Step 5: Manager receives approval notification
        → Manager approves or rejects within 2 working days

Step 6: For upgrade / additional device:
        → IT Head reviews and approves within 3 working days

Step 7: Upon approval, IT procures / configures the laptop
        → Standard provisioning time: 5–7 working days

Step 8: IT notifies employee when laptop is ready
        → Collect from IT Help Desk (Floor 2)
        → Sign IT Asset Acknowledgement Form

Step 9: For replacements, return the old device to IT
        → IT conducts data wipe before reassignment or disposal
```

## 5. Damaged or Stolen Laptop

### 5.1 Accidental Damage

```
1. Report to IT Help Desk immediately: helpdesk.company.com or Ext. 1100
2. IT assesses the damage:
   a. Minor (screen crack, keyboard issue) → Repair sent to vendor (3–5 days)
   b. Major (motherboard failure) → Replacement issued
3. If damage is due to employee negligence, cost recovery may apply:
   - First incident: Warning
   - Second incident: Partial cost recovery (up to 25% of device value)
   - Third incident: Full cost recovery
```

### 5.2 Stolen Laptop

**This is a security incident. Act immediately.**

```
Step 1: Report to your manager and IT Security IMMEDIATELY
        → Email: security@company.com
        → Phone: Ext. 1200 (IT Security)

Step 2: IT Security remotely wipes the laptop via Microsoft Intune

Step 3: File a police complaint (FIR) within 24 hours
        → Submit FIR copy to HR and IT within 3 working days

Step 4: IT raises a replacement request on your behalf

Step 5: Employee's accounts are reviewed and secured (all tokens revoked)
```

## 6. Laptop Return Process

Employees must return company laptops in the following situations:

- Resignation / termination
- End of contract (for contractors)
- Device upgrade (old device must be returned)
- Long leave (more than 3 months)

### Return Process

```
1. Backup all personal files to your OneDrive / approved storage
   (Company data must NOT be copied to personal drives)
2. Visit IT Help Desk (Floor 2) with the laptop and all accessories
   (Charger, laptop bag, docking station if issued)
3. IT performs data wipe and verifies all accessories
4. Sign the IT Asset Return Form
5. IT issues a "Device Returned" clearance certificate
   (Required for Full & Final settlement)
```

## 7. Laptop Care Guidelines

| Do | Don't |
|----|-------|
| Use the original charger and cable | Don't use third-party chargers not approved by IT |
| Keep the laptop in a protective bag | Don't leave the laptop in a car or exposed to extreme heat |
| Clean the screen with a microfiber cloth | Don't use cleaning sprays or wet wipes on the keyboard |
| Report damage to IT immediately | Don't attempt self-repair or open the laptop |
| Lock the screen when stepping away | Don't share your login credentials |
| Keep the OS and software updated | Don't disable automatic security updates |

## 10. FAQs

**Q: How do I request a new laptop?**  
A: Raise a request at `helpdesk.company.com` → New Request → Hardware → Laptop Request. Your manager will need to approve before IT can process it.

**Q: My laptop screen is cracked. What should I do?**  
A: Raise a ticket at `helpdesk.company.com` under "Hardware → Laptop Damage". IT will arrange repair or replacement based on the assessment.

**Q: How long does it take to get a replacement laptop?**  
A: For critical cases (you have no working device), IT prioritizes replacement within 24–48 hours. Standard replacements take 5–7 working days.

**Q: Do I need to return my old laptop when I get an upgrade?**  
A: Yes. The old device must be returned to IT when you collect the new one. IT will wipe and reassign or responsibly dispose of it.

**Q: My laptop was stolen. What should I do first?**  
A: Immediately contact IT Security at Ext. 1200 or email security@company.com. IT will remotely wipe the device and secure your accounts. Then file a police complaint (FIR) within 24 hours.
